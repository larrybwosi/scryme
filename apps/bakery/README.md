# Scryme Bakery Desktop App

This is a standalone Tauri-based desktop application for managing bakery production.

## 🚀 Local Mode

The app supports a fully functional local-only mode that doesn't require an internet connection or a Scryme Cloud account.

### 🔐 Default Credentials

When using Local Mode, you can use the following default administrative credentials to log in:

- **Email:** `admin@bakery.local`
- **Password:** `admin123`

### ⚙️ Changing Credentials

Once logged in, you can change these default credentials by going to:
**Settings > Security > Local Admin Credentials**

## 🌐 Cloud Mode

To sync your data with the Scryme Cloud, you must provision the device with a setup token from your Scryme dashboard. Once provisioned, the app will automatically sync recipes, inventory, and batches.
